# Data Visualization Task

This repository contains a basic data visualization task completed using Python, Pandas, Matplotlib, and Jupyter Notebook.

## Objective

The objective of this task is to create bar charts, line charts, and pie charts from a student dataset.

## Visualizations Created

- Bar chart showing student marks
- Line chart showing the marks trend
- Pie chart showing course distribution
- Attendance comparison chart
- Marks and attendance comparison chart

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git
- GitHub
